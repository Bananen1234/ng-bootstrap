Badges

Use a badge .badge together with a badge modifier .badge-* to highlight or categorize something.

Add .badge-close together with a close button to get a badge that can be dismissed.

Javascript needed! Please note that the example below is just a representation of how dismissable badges look and the needed markup. In order to be able to dismiss or remove them you need javascript to control when and how to show them.
