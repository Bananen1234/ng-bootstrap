Button groups

Group buttons together using a button group, check bootstrap docs for more info and examples. Add .active class to a button to highlight it as active. Note that javascript is required to use button groups like radio buttons or checkboxes.

Disabled state

Triggered by .disabled class or disabled attribute on each button.
