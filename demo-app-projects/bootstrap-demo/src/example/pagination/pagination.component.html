Pagination
Javascript needed! Please note that the example below is just a representation of how the pagination looks and the needed markup for it. In order to use pagination you need a framework to control when and how to show different pages. If you're using angular, please take a look at ng-bootstrap and their pagination component.

Add .pagination-sm or .pagination-lg modifier class to the base .pagination class to adjust the size.

Use a grid to align the pagination controls nicely with other controls or information related to whatever is paginated.
