Spinners

Loading spinner to indicate that something is loading.

Loading spinners can be used in buttons to indicate that an action has been executed and that we're waiting for something to happen before enabling the button again.

Use utility class for background color .bg-* to spinner element when placed against other backgrounds.

Light background .bg-light

And so on... spinners can be used on any solid background by adding a bg-* class either to the spinner element itself or to it's parent container. See available bg classes under the colors section.
